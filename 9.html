HTML ELEMANLARI BULMA

  1. HTML öğelerini kimlik numarasıyla bulma
  2. HTML öğelerini etiket adına göre bulma
  3. HTML öğelerini sınıf adına göre bulma
  4. CSS seçicileriyle HTML öğelerini bulma
  5. HTML öğelerini HTML nesne koleksiyonlarıyla bulma

1. Id ile HTML Elemanı Bulma

JS dosyasına alttaki kodu yazarak bu sayfadan id = "test olanları çağırabilirsin.

document.getElementById("test");

// dükümanda id si test olan değeri buraya yaz demek

2.Etiket(tag) Adına Göre HTML Elements Bulma

JS dosyasına alttaki kodu yazarak bu sayfadan tag ı p olanların hepsini çağırabilirsin.

document.getElementsByTagName("p");

// dükümanda etiketi p olan değeri buraya yaz demek

3.HTML Öğelerini Sınıf Adına göre Bulma

JS dosyasına alttaki kodu yazarak bu sayfadan class ı = "sınıf" olanların hepsini çağırabilirsin.

document.getElementsByClassName("sınıf");

// dükümanda class i sınıf olan değeri buraya yaz demek

4.HTML DOM querySelector () Yöntemi

querySelector() yöntemi, belgede belirtilen bir CSS seçiciyle eşleşen ilk öğeyi döndürür.
Not: querySelector() yöntemi yalnızca belirtilen seçicilerle eşleşen ilk öğeyi döndürür.

Tüm eşleşenleri geri getirmek için bunun yerine querySelectorAll() yöntemini kullanın..

document.querySelector(CSS selectors)

querySelector () nesne kolaksiyonlarına agöre daha hızlıdır.
