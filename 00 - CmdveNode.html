CMD KULLANIMI

  1. Cmd komut istemini aç
    2. - Başlat --> Cmd (komut istemi) yaz
    = Tıkla açılsın...
  2. Uygulama yüklü mü? Kontrol sağla...
    3. - Uygulama adı + boşluk + -- + version yaz.... Enterla
    = node --version
    Yüklü olduğunu söylüyor ve versiyonunu yazıyor.
  3. Uygulama/dosya içinde çalışma yap...
    4. - Uygulama adını yaz... Enterla
    = node
    nursa kullanıcısının içindeki node uygulamasına bağlan dedik. Artık o dosya içinden çalışıyoruz.
    Dosya içinde olduğumuzu > bu işaretten anlıyoruz.
  4. Uygulamadan çıkmak için "Ctrl D" ye tıkla
    5. - İçinden çıktı.
    1 -başka bir dosya açabilirsin.
    = Başa dön adımları tekrarla.
    2- Cmd den çıkış yapabilirsin.
  5. Cmd yi kapatmak istiyorsun.
    6. - "exit" yazabilirsin. (Çarpılayarak da kapatabilirsin.)
